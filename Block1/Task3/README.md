При выполнении passByValue функция принимает массив через параметр и ничего не меняет, поэтому он передается по ссылке и ресурсы компьютера задействуются по минимуму
При выполнении passByValueAndModify массив изменяется, поэтому передается значение, что увеличивает нагрузку на рерурсы компьютера;
При выполнении passByReference передается ссылка и изменение массива не сильно задействует ресурсы компьютера;
Можно сделать вывод работы copy on write, при изменении происходит копирование, что позволяет оптимизировать работу программы
