При запуске БЕЗ declare(strict_types=1)
    - строки складываются, выполняется динамическое привидение типов, но в консоли выдается предупреждение Depricated;
    - цифры с плавающей запятой складываются, выполняется динамическое привидение типов к int, в консоли также Depricated;
    - при сложении Int и null происходит ошибка TypeError;
При запуске С declare(strict_types=1)
    - строки не складываются, ошибка TypeError;
    - переданные float не совместимы с указанными int параметрами, TypeError;
    - переданный null параметр как определенный int вызывает ошибку TypeError;
